외부스타일 시트 작성 후 적용순서

  1. style폴더 내에 파일명.css로 css파일을 생성함
  2. 연결하려는 HTML파일에 head태그 내에 link태그로 위 1번을 연결함
  3. 포토샵 및 피그마에 맞춰서 HTML을 작성 후 저장함
  4. 1번 css파일로 돌아가서 원하는 선택자와 속성을 작성하며 디자인하고 저장함
  5. HTML로 돌아가서 한번 더 저장 후 Go Live 이용해서 실시간 미리보기 함

CSS

CSS 속성과 값 작성방법

속성 주의사항

  • 영문소문자 작성
  • 공백불가, 특수문자 불가, 하이픈작성

값 주의사항

속성과 값 끝날 시 ;(세미콜론)작성필수

CSS 속성-색상

css에서 작성할 수 있는 색상 값의 종류

color, background-color

  • 헥사코드
  • #으로 시작하는 6자리 RGB코드 ex)#ffff00
RGBA코드
RGB와 Alpha(A) 투명도를 함께 적용할 수 있는 코드
ex)rgba(255,0,0,0.2) 빨강색 20% 투명도

선택자 적용 우선 순위

인라인 > ID > Class > Tag
위 순서대로 우선순위가 적용됨
서로 다른 선택자 간의 동일한 속성이 서로 다른 값으로 되어있을 때 우선순위에 따라 순위가 높은대상의 값이 적용됨
